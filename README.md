# Multi-Robot-SLAM-and-Autonomous-Navigation-System

> Collaborative mapping system enabling two TurtleBot3 robots to simultaneously 
> explore and merge maps in ROS/Gazebo simulation

**Features:**
- Namespace-isolated multi-robot operation
- Real-time map fusion from independent SLAM processes
- AMCL localization + move_base navigation
- RViz visualization of multi-robot sensor data
